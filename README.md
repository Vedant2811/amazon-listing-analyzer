# Amazon Listing Analyzer 🔍

An AI-powered tool that analyzes Amazon product listings and gives actionable optimization suggestions.

## What it does
- Paste any Amazon product URL or ASIN
- Instantly get scores for Title, Bullet Points, and Keywords
- See top issues hurting your listing
- Get quick wins and an AI-improved title suggestion

## Built with
- Node.js + Express
- Google Gemini AI
- Vanilla HTML/CSS

## Why I built this
Built this as a demo project relevant to Pixii's domain — helping sellers optimize their Amazon listings with AI.

## Run locally
git clone https://github.com/Vedant2811/amazon-listing-analyzer
cd amazon-listing-analyzer
npm install
# Add your GEMINI_API_KEY to .env
node server.js
